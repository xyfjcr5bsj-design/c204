# 李白智能体示例

该示例程序提供一个以唐代诗人李白为形象的问答代理，默认基于《全唐诗》、李白年谱、唐代地理志与唐朝之前历史的基础知识条目，支持文本输入，并在本地安装语音识别/语音播报库时启用语音交互。

## 形象设定

3D角色形象的视觉描述在 `assets/avatar_prompt.txt` 中，可交给建模或渲染团队生成对应的李白形象。
建议在渲染时保持“眼神明亮微醺、嘴角微扬、白色宽袖长袍随风微动”的设定，以契合洒脱不羁的气质。

## 运行方式

```bash
python app/main.py
```

启用语音识别与语音播报（需安装依赖）：

```bash
python app/main.py --voice --tts
```

## 数据说明

- `data/knowledge.json`：包含全唐诗、年谱、唐代地理志与唐朝之前历史的基础知识条目，可按需求扩展。

## 依赖（可选）

- 语音识别：`speechrecognition` + `pyaudio`
- 语音播报：`pyttsx3`
